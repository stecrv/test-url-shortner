# Runnig/Testing the code

## Install

* Open terminal,
* Go to root folder
* `npm install`

## Run

* Open terminal
* `npm run start` or `node app.js`

## Test

* Open terminal
* `npm run test` or  `mocha ./tests/`

## Front end page

* Run the application
* Oper browser with URL "http://localhot:4000/index"
